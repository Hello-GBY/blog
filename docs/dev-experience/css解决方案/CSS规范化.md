EM 是由 Yandex 团队提出的一种 CSS Class 命名方法，旨在帮助开发人员创建更好的且结构一致的 CSS 模块。

CSS 规则很可能互相干扰（选择器权重问题、命名冲突问题），带来混乱
由于不确定会发生什么，无人敢删 CSS 代码。于是项目里的 CSS 代码永远只增不减，成为一个烂摊子

extends字段为官方推荐的标准规则
需要覆盖官方规则可在rules字段中配置
具体配置可查阅[官方文档](url)
